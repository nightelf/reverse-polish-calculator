# reverse-polish-calculator
A command line reverse polish calculator

## Requirements:
* PHP 7
* Composer
* PHPUnit

## Command line instructions:
Clone the repository and run these commands from the repository root directory:
1. composer update
2. phpunit
3. php run.php

## Console app commands:
* q : quit
* c : clear the stack
* s : show the stack
* \+ : add
* \- : subtract
* \* : multiply
* / : divide
* ** : exponent
* i : inverse
